 Project Description

This project presents the design and implementation of a 4-wheel autonomous mobile robot based on the ESP32 microcontroller. The platform uses a differential drive architecture with independent motor control and wheel encoder feedback, enabling precise motion control and autonomous navigation capabilities.

The system integrates embedded control, motor driving, and real-time feedback acquisition to provide a robust foundation for robotics experimentation, including navigation algorithms, closed-loop control, and intelligent perception.

The ESP32 serves as the main controller, generating PWM signals to drive the motors via an H-bridge motor driver, while encoder feedback enables accurate measurement of wheel speed and motion estimation. The mechanical structure is built on a metal chassis with four DC gear motors, ensuring stability and traction.

This project provides a complete embedded robotics platform suitable for developing and testing:

- Autonomous navigation algorithms  
- Closed-loop motor control (PID)  
- Odometry and motion estimation  
- Obstacle avoidance systems  
- Embedded AI and robotics applications  

The modular architecture allows easy integration of additional sensors such as ultrasonic sensors, cameras, IMU, or LiDAR for advanced autonomous functionalities.

This platform serves as a practical foundation for robotics research, embedded systems development, and autonomous vehicle prototyping.
